Getting Started

  • Select your preferred search method in the navigation menuleft sidebar
  • Enter the required search fields, marked with an *.
  • The search results are displayed in a new card format. You can filter by name, document type, recording/plat book and page, date, and address.
  • Selecting a record (card) will display the document details.
  • To order a certified copy, navigate to the document details and select Order Certified Copies
  • To use advanced search you need to register/login.
  • Please help us improve our website by providing feedback.

Key Features

  • Records Search
     Users can search for official records utilizing our modernized standard and advanced searches.
  • Public Access
     The system ensures transparency by providing public access to official records.
  • Registered Access
     Avoid reCaptcha and search limitations by registering and/or logging in.
  • ADA Compliant
     Designed to ensure equal access for individuals with disabilities.
